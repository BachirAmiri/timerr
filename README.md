# Timerr App

Timerr is time tracking app designed for mobile use

Using [Ionic](https://ionicframework.com/) with [Angular](https://angular.io/) and [Cordova](https://cordova.apache.org/)

## Run

```bash
npm install
npm start
```

This will open by default on localhost:4200
